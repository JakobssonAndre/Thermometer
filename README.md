# Thermometer

Program for displaying temperature on LCD using Arduino.

## Hardware

- Keyestudio Mega 2560 R3   (SBC)
- 1602A                     (LCD)
- NTC-MF52 3950             (Temperature Sensor)

## Software

- Visual Code               (IDE)
- PlatformIO                (Platform)
- NTC_Thermistor            (Library)
- LiquidCrystal             (Library)
